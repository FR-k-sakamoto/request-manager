# Security Best Practices

このスキルは、主要なWebフレームワークに対してセキュリティベストプラクティスの
レビューやセキュア・バイ・デフォルトな実装ガイダンスを提供します。

## 使うタイミング

- 既存コードのセキュリティレビューや脆弱性指摘をしたい
- セキュアな実装方針の確認をしたい
- セキュリティ上の注意点を洗い出したい

## 対応範囲

- Python: Django / Flask / FastAPI
- JavaScript/TypeScript: Express / Next.js / React / Vue / jQuery
- Go: 一般的なバックエンド

## 必要な入力

- 対象リポジトリ / ファイル
- 使用言語・フレームワーク（不明なら自動判定）

## 出力

- 重要度付きの指摘（必要に応じて）
- 具体的な修正方針
- 明示的な監査依頼時は`security_best_practices_report.md`を作成

## 注意点

- 通常のコードレビューではなく、セキュリティ目的の依頼時に使用する
- 対応外の言語・FWでは一般的な指針のみになる場合がある
